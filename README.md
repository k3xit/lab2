# Поморцев Аким КМБ-1 Лабораторная №2

# Задание 1

## Задача 1

### Текст задачи

Получить список из первых цифр чисел, содержащихся в исходном списке

### Алгоритм решения

Пользователь поочередно вводит целые элементы списка, проверяем на правильность ввода, при ошибке выводим сообщение об ошибке и повторяем ввод. Выводим на экран изначальный список. С помощью List.map каждый элемент списка отправляем в рекурсивную функцию, если при делении на 10 целая часть равна нулю, возвращаем число, иначе вызываем функцию с числом, деленным на 10. В итоге формируется список из первых чисел исходного списка и выводится на экран. Программа завершает работу.

# Задание 2

## Задача 1

### Текст задачи

Сколько раз в списке встречается заданное число?

### Алгоритм решения

Пользователь поочередно вводит целые элементы списка, проверяем на правильность ввода, при ошибке выводим сообщение об ошибке и повторяем ввод. Пользователь вводит число для сравнения, проверяем на правильность. Выводим на экран получившийся список. С помощью List.fold проходим по списку и сравниваем элементы с искомым, если они равны, увеличиваем acc на один и возвращаем, иначе возврщаем неизменный. Выводим результат функции на экран. Программа завершает работу.
